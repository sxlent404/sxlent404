# Sxlent404's Slap Battles Script Hub 👏

## Welcome! 👋

This is an open-source Roblox script hub dedicated to enhancing your Slap Battles experience across various game modes and places within the Slap Battles ecosystem.

### 🌟 Features

- Support for multiple Slap Battles game modes and places
- User-friendly GUI powered by OrionLib
- Silent execution option for seamless gameplay
- Config saving functionality to preserve your settings

### 🎮 Supported Slap Battles Places

- Most of the places in Slap Battles! 👏

### 📥 Installation

To use this script, copy and paste the following loadstring into your executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/sxlent404/sxlent404/main/slapbattles.lua"))()
```

### 🛠️ Usage

1. Execute the script in any Slap Battles place.
2. Use the GUI to access various hacks and features specific to each game mode.
3. Enable "Silent Execute" for automatic script execution when joining or teleporting between Slap Battles places.

### 🔄 Auto-Update

This script automatically fetches the latest version when executed, ensuring you always have the most up-to-date features and fixes.
